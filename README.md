# parcelVueApp

#### Description
Blazing fast, zero configuration vue application bundler.

#### Installation

```
npm install
```

#### Instructions

1. Compiles and hot-reloads for development
```
npm run serve
```

2. Compiles and minifies for production
```
npm run build
```
